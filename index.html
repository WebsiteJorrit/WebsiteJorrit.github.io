Wat ik doe

Ik ben een laatstejaarstudent burgerlijk ingenieur computerwetenschappen aan de KU Leuven. Ik hou van het ontwikkelen van software en het oplossen van problemen. Daarnaast ben ik erg vertrouwd met web development, gaande van simpele landing pages tot complexe webapplicaties. Een andere passie van mij is het ontwerpen van machine learning applicaties. Dit was dan ook mijn afstudeerrichting en in deze tak heb ik ook al enkele studentenjobs gedaan. In mijn vrije tijd denk ik graag na over startups, waaruit dan ook al enkele MVP's zijn voortgekomen.

Burgerlijk ingenieur computerwetenschappen

Gedurende mijn opleiding hebben wij enorm veel software moeten ontwikkelen, gaande van chess engines in C++ over webapplicaties in Javascript tot machine learning applicaties in Python.

Web developer

Dit gaat van simpele landing pages tot complexe webapplicaties ontwikkeld in React of Angular. Voor diverse MVP's heb ik backend technologieën zoals Strapi, Flask & Firebase gebruikt.

Machine learning engineer

Vanwege mijn afstudeerrichting heb ik veel ervaring met het ontwerpen van machine learning applicaties. Een voorbeeld hiervan is een 'Visual Question Answering' chatbot die ik heb ontwikkeld in PyTorch. Een ander voorbeeld is een 'Image segmentation' model die treinsporen detecteert.
