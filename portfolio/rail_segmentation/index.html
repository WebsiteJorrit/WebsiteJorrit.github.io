Een Machine Learning model die sporen kan segmenteren

  • Categorie: Machine Learning
  • Talen & Frameworks: Python, PyTorch
  • Datum: 21 April 2021

Voor mijn studentenjob die ik uiteindelijk in de zomer van 2021 ging doen bij OTIV, had ik de opdracht gekregen om een Machine Learning model te maken die sporen kan segmenteren. Als basis gebruikte ik het U-model, wat ik van scratch heb geimplementeerd in PyTorch. Na het instellen van de Dataloaders en het trainen van het model heb ik een presentatie gegeven waarna ik mijn studentenjob bij OTIV heb mogen doen.
