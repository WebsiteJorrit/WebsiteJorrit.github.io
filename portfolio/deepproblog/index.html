Neuro Symbolic AI - Visual Question Answering met behulp van DeepProbLog

  • Categorie: Machine Learning
  • Talen & Frameworks: Python, PyTorch, DeepProbLog
  • Datum: 23 April 2022
  • Source code: Github link

Voor het vak 'Capita Selecta AI' heb ik gekozen om me verder te verdiepen in Neuro Symbolic AI. Specifiek heb ik hiervoor DeepProbLog gebruikt, waarbij probabilistisch logisch denken gecombineerd wordt met neurale netwerken. In dit project heb ik ook veel met PyTorch gewerkt voor het opzetten van de neurale netwerken en de data loaders. Het doel van dit project was om een Visual Question Answering model te maken, waarbij het vragen moet kunnen beantwoorden gebaseerd op een afbeelding. Ik heb ook een paper geschreven over dit project, die je hier kan vinden.
